# ICE (Interactive Connectivity Establishment)

# 1. Gathering Candidate Addresses

# Candidate是什么

​ A CANDIDATE is a transport address -- a combination of IP address and port for a particular transport protocol (with only UDP specified here).

# Candidate类型

  • 主机候选者（本机所有IP和指定端口）
  • 反射候选者（NAT上的IP和端口）
  • 中继候选者

# 发送给对方

一方收集到所有的候选者后，通过信令传给对方，格式为sdp，当双方都拿到全部列表后，将候选者形成匹配对

# 2. Connectivity Checks

  1. Sort the candidate pairs in priority order.
  2. Send checks on each candidate pair in priority order.
  3. Acknowledge checks received from the other agent.
