# TURN (Traversal Using Relay NAT)

​ TURN (Traversal Using Relay NAT) 是一种网络协议，其目的时解决对称NAT无法穿越的问题。与STUN协议不同，TURN协议通过**中转服务器（relay server）**来传输数据，以便在NAT后面的设备之间实现直接通信。

​ TURN是建立在STUN之上，消息格式使用STUN格式消息
